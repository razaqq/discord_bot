# discord_bot

Dependencies:
  - PrettyTable
  - discord.py >= 1.0.0
  - matplotlib
  - sqlite3
  - tweepy
  - requests
  - json
  
Tested only on python >= 3.7.0
 
Install:
  - git clone https://github.com/razaqq/discord_bot.git
  - start the bot with "python bot.py"
  - edit config file under /config/config.py
  - start the bot again
